# Multimedia Editing Portfolio

This repository contains sample work demonstrating my experience with open-source audio and video editing tools. The goal of this portfolio is to showcase basic multimedia editing workflows including audio cleaning, video trimming, subtitle editing, and simple music production.

Tools Used

• Audacity  
• Shotcut  
• OpenShot  
• LMMS  

---

## Audio Editing – Audacity

Example tasks performed:

• Noise reduction  
• Voice trimming  
• Volume normalization  
• Exporting audio files  

Workflow example:

1. Imported raw voice recording  
2. Applied noise reduction filter  
3. Trimmed silent sections  
4. Adjusted volume levels  
5. Exported cleaned audio  

Sample file: `cleaned_voice_sample.mp3`

---

## Video Editing – Shotcut

Example tasks performed:

• Video trimming  
• Adding transitions  
• Basic color adjustment  
• Exporting video to MP4  

Workflow example:

1. Imported short video clip  
2. Trimmed unnecessary sections  
3. Added simple transition effects  
4. Exported final video in 1080p  

Sample file: `edited_video_demo.mp4`
